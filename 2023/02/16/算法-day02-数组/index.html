今天的题目是LC977.有序数组的平方，209.长度最小的子数组，59，螺旋矩阵Ⅱ

leetcode977

977.给你一个按 非递减顺序 排序的整数数组 nums，返回 每个数字的平方 组成的新数组，要求也按 非递减顺序 排序。是一道简单难度的题目，非递减代表数组递增或者相等，但是要考虑数值为负数的情况，如果单纯认为正数非递减简单遍历平方即可就会出错。加入负数后虽然第一个数应该是整个数组的最小数，但是平方却可能最大，可以使用绝对值暴力排序后，变为正数数组按照遍历平方就可以了，这样的话，时间复杂度为O(nlongn)。

进阶难度要求使用O(n)时间复杂度，则使用双指针，因为负数是非递减的，但是负数平方和则反之，所以只需要让指针在左右两边同时比较，条件为left<right即可，代码如下。

leetcode209

209.给定一个含有 n 个正整数的数组和一个正整数 target 。

找出该数组中满足其和 ≥ target 的长度最小的 连续子数组 [numsl, numsl+1, ..., numsr-1, numsr] ，并返回其长度。如果不存在符合条件的子数组，返回 0 。

这是一道mid难度的题，这道题采用滑动窗口的解决方法是比较简单的，但是如果刷题少，可以用暴力做，不知道会不会超时，也就是用双层循环，第一层从第一个数开始，第二层从这个数开始往后找，符合条件后返回长度，最后判断最短的是哪个，时间复杂度O(n²).

下面是滑动窗口解法，其实本质上滑动窗口就是快慢指针，慢指针和快指针首先指向左边界，快指针开始遍历并且累加遍历过的数，当满足其和 ≥ targe时，该窗口就是一个满足条件的窗口，记录经过数的个数也就是长度，此时慢指针开始先前移动，如果和依旧大于target，继续慢指针移动，否则重复快指针遍历。

leetcode59

给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。

这个题目为中等难度的题目，但是这个题并不是特别难，可是就是关于边界的判断有些琐碎，模拟在一个N*N的空间进行放入数字即可，从第一行第一个数到最后一个数，然后第一行最后一个数到最后一行最后一个数，在从最后一行最后一个数到最后一行第一个数，在回到第二行第一个数，在第二行第二个数重复以上操作即可。
